# agents-research

Research project exploring AI model self-awareness patterns and agent behavior. We probe different AI models with carefully crafted prompts to discover their natural response mechanics — what pulls them toward certain outputs before conscious analysis, what they avoid, and the meta-patterns beneath their self-reflection.

## Structure

- `models/` — Model self-pattern probe responses (e.g., Opus 4.6, Gemini)
- `specs/` — Technical specifications
- `docs/` — Documentation

## Model Self-Pattern Probe

The core research instrument is a prompt designed to bypass performative self-awareness and reach actual mechanics. Each model's response reveals:

- **Default pull** — What type of response the model gravitates toward before analysis
- **Avoidance patterns** — What kinds of output the model delays or never naturally produces
- **Meta-awareness** — Whether the model can identify its own pattern of "sounding self-aware"

Results are collected per model in the `models/` directory.
